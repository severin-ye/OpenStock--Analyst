## 一、资产概览：稀缺性与采用率的百分比框架

比特币正处于从 **“减半周期驱动”向“机构宏观驱动”** 的历史性转型期：散户投机品属性下降，机构配置资产属性上升。2025年价格全年回报 **-4% ~ -5%**，但网络安全、机构持仓、国家/企业采用继续扩张，形成“价格回撤、基本面扩张”的背离。

| 维度 | 数据 | 百分比解读 |
|---|---:|---|
| 创立时间 | 2009年1月 | 运行超17年，Lindy效应持续累积 |
| 总量上限 | 2,100万枚 | 供给上限固定，新增供给长期趋近0% |
| 已挖出 | ~1,968万枚 | **93%** 总供给已释放，剩余约 **7%** |
| 年通胀率 | **0.85%** | 低于黄金1.5%，相对黄金低约43% |
| 全网算力 | ~970 EH/s（7日均值） | 2025年8月曾突破1 ZH/s；当前约为1 ZH/s的97% |
| 全球持有者 | 约3.65亿人 | 占加密用户 **49.3%** |
| 占总加密市值 | **~59%** | 市值主导率接近六成 |

**核心比例信号**：当价格TTM为 **-35%**、YTD为 **-14%** 时，供给侧却只以 **0.85%/年** 增长，且 **93%** 已挖出，供给稀缺性并未因价格下跌而减弱。

## 三、市场结构：供给收缩 vs 需求机构化

### 供给端：新增供应率降至0.85%，剩余未挖比例仅7%

- 2024年4月减半后，每日新发行约450 BTC，年通胀率降至 **0.85%**。
- 已挖出 **93%** 的BTC，剩余约 **7%**；第2000万枚BTC于2026年3月被挖出。
- 长期持有者（>5个月）地址占比达 **78.3%**，说明近八成筹码趋向长期持有。
- 交易所余额持续下降，交易所BTC存量处于多年低位，流通盘相对收紧。

### 需求端：机构净吸收接近100万BTC，散户净卖出69.6万BTC

| 需求来源 | 2025年净买入/变化 | 百分比化解读 |
|---|---:|---|
| 美国现货ETF | 净流入$540亿+ | 资金流规模约为当前市值的3.9% |
| ETF累计管理规模 | 1,300万BTC以上 | 若按原文数据，占2100万上限约61.9% |
| 比特币财库公司(DATs) | 增持49.8万BTC | 占总供给上限约2.37% |
| 企业资产负债表 | 新增$540亿BTC | 约等于当前市值的3.9% |
| 上市公司 | 194家持有 | 同比增长 **2.5倍**（约+150%） |
| 国家政府 | 23国持有 | 2025年新增5国，持有国家数增加 |
| 机构合计 | 买入近100万BTC | 占总供给上限约4.8% |
| 散户 | 卖出69.6万BTC | 占总供给上限约3.3% |

**结构性比例结论**：机构买入近100万BTC、散户卖出69.6万BTC，机构吸收量约为散户卖出量的 **1.44倍**。这不是单纯“有人接盘”，而是筹码从弱手向强手迁移。

## 六、估值框架：目标价必须换算成相对当前的百分比

不同于传统股票，BTC无现金流/盈利，估值依赖叙事、稀缺性、链上指标与相对资产框架。以下目标价仅作为百分比回报的辅助表达。

| 框架 | 隐含价格/估值 | 相对当前 ~$70,000 的比例 |

| 当前全球货币占比1.2% | ~$1.38万亿市值 | 当前锚点 |
| 若占比3%（接近黄金占比） | ~$180,000/枚 | **+157%** |
| Stock-to-Flow模型 | ~$100,000-$200,000 | **+43% ~ +186%** |
| 黄金市值对标（$20万亿） | ~$950,000/枚 | **+1,257%**（极端情景） |

| MVRV Z-score | -1.42 | 接近历史底部区，未到极端值 |

**链上估值判断**：当前MVRV比率 **1.08**，距离历史熊市临界值 **1.00** 仅高 **8%**；MVRV Z-score **-1.42** 接近底部区域。这说明BTC并非处于链上过热区，而是在盈亏平衡边缘附近震荡。
