# 回测实验对比报告：单一排名 vs 集成分析模块

> **实验日期**: 2026-05-28
> **实验标的**: 英伟达 (NVDA) · 小米 (1810.HK) · 特斯拉 (TSLA)
> **回测窗口**: 2023-05 至 2025-11（3年，6个有效时点）

---

## 一、实验背景

### 1.1 研究目的

本实验旨在对比两种预测方法的有效性：

1. **单一排名系统**：仅使用 Greenblatt 四层加权排名（EBIT/EV + ROIC + F-Score + PEG）
2. **集成分析系统**：在排名基础上，集成技术分析、内部人交易、机构持仓等多个维度

### 1.2 核心问题

- 集成分析模块是否比单一排名更准确？
- 技术分析能否纠正基本面排名的错误判断？
- 多维度信号综合后的预测效果如何？

### 2.3 信号生成逻辑

#### 方法 A：单一排名系统

```python
# 直接从排名位置判定信号
if rank_num <= total // 3:
    signal = "BULLISH"
elif rank_num <= total * 2 // 3:
    signal = "NEUTRAL"
else:
    signal = "BEARISH"
```

#### 方法 B：集成分析系统

```python
# 1. 排名信号（同方法 A）
rank_signal = get_signal_from_rank(rank_result)

# 2. 技术分析信号
tech_signal = TechnicalAnalyzer().analyze(ticker)

# 3. 内部人交易信号
insider_signal = InsiderAnalyzer().analyze(ticker)

# 4. 机构持仓信号
inst_signal = InstitutionalAnalyzer().analyze(ticker)

# 5. 综合投票
final_signal = vote([rank_signal, tech_signal, insider_signal, inst_signal])
```

## 五、核心发现

### 5.1 集成分析显著提升预测准确性

| 指标 | 单一排名 | 集成分析 | 提升 |
|------|:-------:|:-------:|:----:|
| 6月正确率 | 23.1% | **61.5%** | **+38.4%** |
| 1年正确率 | 40.0% | **60.0%** | **+20.0%** |

### 5.2 技术分析是最有价值的补充模块

- 技术信号正确率 55.6%，高于排名信号的 23.1%
- 技术分析在英伟达和特斯拉上纠正了排名系统的错误判断
- 当排名和技术冲突时，技术分析往往更准确

### 5.3 NEUTRAL 信号避免了错误判断

- 当信号冲突时，系统给出 NEUTRAL
- NEUTRAL 信号避免了在不确定情况下的错误推荐
- 这是集成系统的重要优势

### 5.4 小米的预测仍然困难

- 排名系统和技术系统都给出保守信号
- 但小米实际持续上涨（EV 叙事 + AI 概念）
- 可能需要额外的"叙事分析"模块

## 六、改进建议

### 6.1 短期改进

1. **优化技术分析权重**：当技术信号与排名信号冲突时，给予技术分析更高权重
2. **添加更多技术指标**：MACD 金叉/死叉、RSI 超买/超卖
3. **改进 NEUTRAL 信号**：当 NEUTRAL 时，明确说明原因

### 6.2 中期改进

1. **集成更多分析模块**：
   - 财报电话会议分析（管理层情绪）
   - 行业轮动分析（经济周期）
   - 竞争分析（护城河）

2. **动态权重调整**：
   - 根据历史正确率动态调整各模块权重
   - 对不同公司使用不同的权重组合

### 6.3 长期改进

1. **机器学习集成**：
   - 使用历史数据训练信号权重
   - 实现自适应学习

2. **叙事分析**：
   - 分析新闻、社交媒体情绪
   - 捕捉"EV 叙事"、"AI 概念"等市场主题

## 七、结论

### 7.1 实验结论

1. **集成分析系统显著优于单一排名系统**：6月正确率从 23.1% 提升到 61.5%
2. **技术分析是最有价值的补充**：纠正了排名系统在高增长股上的错误判断
3. **NEUTRAL 信号是重要创新**：避免了在信号冲突时的错误推荐

### 7.2 使用建议

| 场景 | 建议 |
|------|------|
| 排名 + 技术一致 BULLISH | 强烈推荐买入 |
| 排名 + 技术一致 BEARISH | 强烈建议回避 |
| 排名 BULLISH + 技术 BEARISH | 谨慎，等待更多信号 |
| 排名 BEARISH + 技术 BULLISH | 考虑技术分析，可能被低估 |
