### 1.3 根因分析

#### 根因 1：F-Score 严重拖累

F-Score 是 Piotroski 的财务健康评分（0-9），小米在 2.5yr 和 3.0yr 只有 **1/9** 分。

**9项逐项分析**：

```
盈利组 (4项):
1. ROA > 0              ✅ (取决于年度)  
2. 经营现金流 > 0        ❌ (现金流差)
3. ROA 同比增长          ❌ (ROA下降)
4. 经营现金流 > 净利润    ❌ (应计项目多)

杠杆组 (3项):
5. 长期债务减少          ❌ (持续扩张)
6. 流动比率改善          ❌ (流动性恶化)
7. 股本未稀释            ❌ (持续融资)

效率组 (2项):
8. 毛利率提升            ❌ (竞争压力)
9. 资产周转率提升        ❌ (扩张期效率低)
```

**核心问题**：F-Score 惩罚的是"激进扩张"的公司。小米在 2023 年大举投资 EV，导致：
- 现金流变差（大量资本支出）
- 债务增加（借债投资）
- 资产周转率下降（大量新资产未产生收入）

但市场恰恰看中了这种扩张，给了 EV 叙事溢价。

#### 根因 2：技术分析一致看空

所有时点的 MTF 评分为 **0/3**（强烈看空）。这意味着：
- 股价低于 MA20/MA50/MA200（所有均线）
- 趋势指标全面看空

但小米股价在 2024 年暴涨（+82.9%、+54.0%），**技术指标给出的恰恰是反转前的底部信号**。

#### 根因 3：缺少叙事分析

系统只分析了：
- 历史财务数据 ❌
- 历史价格趋势 ❌

系统没有分析：
- **小米造车叙事**（EV 概念）❌
- **AI 概念**（智能驾驶）❌
- **新闻情绪**❌
- **社交舆情**❌

### 2.2 根因分析

#### 根因 1：EBIT/EV 严重失真

EBIT/EV 是价值投资的黄金指标。高 EBIT/EV 意味着"便宜"。

```
2.5yr (2023-11-27): NVDA EBIT/EV = 0.37%
   → 系统解读：太贵了！
   → 排名 #8/9
   → 信号 BEARISH

但实际回报：+120.7%
```

**为什么 EBIT/EV 失效？**

| 因素 | 解释 |
|------|------|
| **EV 爆炸** | 企业价值 = 市值 + 债务 - 现金。NVDA 市值飙升，EV 被市值拉高 |
| **EBIT 滞后** | EBIT 是历史数据，尚未反映 AI 带来的收入爆发 |
| **分母效应** | EBIT/EV = EBIT / EV。EV 随股价暴涨翻倍，但 EBIT 还没跟上 |

**时序还原**：

```
2023-05: EBIT/EV = 0.46%，EV ≈ $850B，EBIT ≈ $4B
   → 市场在博弈：AI 会让 EBIT 从 $4B 涨到 $20B+
   → 系统只看当前 EBIT，看不到未来

2023-11: EBIT/EV = 0.37%，EV ≈ $1.2T（市值翻倍）
   → EBIT 还没涨，EV 已经翻了
   → 系统认为"越来越贵"
   → 但市场继续涨
```

#### 根因 2：F-Score 惩罚高研发投入

2.5yr 和 3.0yr 的 F-Score 只有 **3/9**。

```
扣分项：
- ROA 同比下降（研发投入增加导致利润增速不如资产增速）
- 流动资产 vs 流动负债恶化（大规模采购 GPU）
- 毛利率没有明显改善（新产品爬坡期）

但这恰恰是 AI 爆发前夜的"蓄力"阶段！
```

#### 根因 3：PEG 数据缺失

2.5yr 和 3.0yr 的 PEG 为 **N/A**（营收增速无法计算），导致：
- L4（增长值不值）失去判断依据
- 无法用 PE 的高估值来论证"增长值这个价"
- 系统更依赖 L1（EBIT/EV）和 L3（F-Score），这两个都给出消极信号

### 3.2 根因分析

#### 根因 1：PE 波动极大

```
2024-05: PE = 38x（相对合理）
2024-11: PE = 71x（暴涨）
2023-05: PE = 48x
2023-11: PE = 59x

PE 从 38x 跳到 71x，系统很难给出稳定的估值判断
```

#### 根因 2：ROIC 和 PE 的背离

```
2.5yr (2023-11): ROIC = 36.06%（极高），PE = 59x（较高）
   → L1 (EBIT/EV): 1.89%（偏贵）
   → L2 (ROIC): 36.06%（优秀）
   → 系统：NEUTRAL（矛盾）
   → 实际：-24.1%（系统正确！）

3.0yr (2023-05): ROIC = 36.06%，PE = 48x
   → L1: 2.32%
   → L2: 36.06%
   → 系统：NEUTRAL
   → 实际：+21.9%（系统错误）
```

#### 根因 3：技术分析给出错误信号

```
2.5yr (2023-11): MTF 3/3 BULLISH → 实际 -24.1%
   → 技术分析在顶部给出买入信号！
   → 这是经典的"追高"陷阱

3.0yr (2023-05): MTF 3/3 BULLISH → 实际 +21.9%
   → 技术分析在底部给出正确信号
```

    # 如果研发/资本支出增长 > 20% 且 ROIC > WACC，加 1 分
    if (capex_growth > 0.2 or rnd_growth > 0.2) and roic > wacc:

    # 如果研发/资本支出增长 < -20% 且 ROIC < WACC，减 1 分
    if (capex_growth < -0.2 or rnd_growth < -0.2) and roic < wacc:
        score -= 1  # 缩减投资减分
    
    return min(9, max(0, score))
